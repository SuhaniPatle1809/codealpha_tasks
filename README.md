# ✨ TalkAcross - Language Translator

A modern, visually appealing desktop language translator built with **Python Tkinter** and **Google Translator API**. TalkAcross allows users to instantly translate text between 12 different languages through an interactive and animated user interface.

## 🌟 Features

* 🌐 Translate text between 12 languages:

  * English
  * Hindi
  * French
  * German
  * Spanish
  * Japanese
  * Chinese
  * Marathi
  * Gujarati
  * Tamil
  * Telugu
  * Punjabi

* 🎨 Modern pastel-themed user interface

* ✨ Animated floating background bubbles

* 🎭 Smooth slide-in entrance animation

* 🔄 Language swap functionality

* 📋 One-click copy translated text to clipboard

* 💫 Interactive button animations and hover effects

* 📱 Clean card-based layout

* ⚡ Fast translations powered by Google Translate

## 🛠️ Technologies Used

* Python 3
* Tkinter (GUI Framework)
* ttk Widgets
* deep-translator
* Google Translator API

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/talkacross-translator.git
cd talkacross-translator
```

### 2. Install dependencies

```bash
pip install deep-translator
```

### 3. Run the application

```bash
python translator.py
```

## 🎯 How to Use

1. Launch the application.
2. Enter the text you want to translate.
3. Select the source language.
4. Select the target language.
5. Click the **Translate** button.
6. View the translated text instantly.
7. Use the **Copy** button to copy the result to your clipboard.
